## About

> **MarkDown++** is a fork of [MarkText](https://github.com/marktext/marktext), an MIT-licensed open-source markdown editor.
> Built upon MarkText with significant enhancements for reading, writing, and AI-assisted editing.

## Features

### Inherited from MarkText
- **Realtime preview (WYSIWYG)** 鈥?Clean and simple interface for distraction-free writing.
- **CommonMark & GFM support** 鈥?Full [CommonMark Spec](https://spec.commonmark.org/0.29/) and [GitHub Flavored Markdown](https://github.github.com/gfm/) support, with selective [Pandoc markdown](https://pandoc.org/MANUAL.html#pandocs-markdown) support.
- **Markdown extensions** 鈥?Math expressions (KaTeX), front matter, emojis, diagrams (Mermaid, PlantUML, Vega).
- **Output formats** 鈥?Export to **HTML** and **PDF**.
- **Themes** 鈥?Multiple built-in themes: Cadmium Light, Material Dark, Graphite, One Dark, etc.
- **Editing modes** 鈥?Source Code mode, Typewriter mode, Focus mode.
- **Clipboard images** 鈥?Paste images directly from clipboard.

### New in MarkDown++

#### :book: ZRead Mode
A dedicated reading mode optimized for documents generated by [ZRead](https://github.com/nlstone/zread) tool. Provides an immersive, distraction-free reading experience with optimized typography and layout.

#### :left_right_arrow: Real-time Split-Pane Preview
Side-by-side live preview of your markdown source and rendered output. Edit on one side, see changes instantly on the other 鈥?the best of both worlds between source code and WYSIWYG editing.

#### :globe_with_meridians: Multi-Language Support (i18n)
Full internationalization with support for multiple UI languages. Currently supports:
- :cn: 绠€浣撲腑鏂?- :us: English
- *(More languages coming soon 鈥?contributions welcome!)*

#### :robot: AI-Powered Editing
Integrated AI assistant for intelligent writing assistance:
- **Smart Rewrite** 鈥?AI-powered content rewriting and restructuring
- **AI Assistant** 鈥?Context-aware suggestions, summarization, and expansion
- **Inline AI** 鈥?Quick AI actions directly in the editor

## Download and Installation

![platform](https://img.shields.io/static/v1.svg?label=Platform&message=Linux-64%20|%20macOS-64%20|%20Win-64&style=for-the-badge)

All binaries for Linux, macOS and Windows can be downloaded from the [release page](https://github.com/nlstone/Markdown-PlusPlus/releases/latest).

#### macOS

Download the latest `markdownpp-%arch%-mac.dmg` from the [release page](https://github.com/nlstone/Markdown-PlusPlus/releases/latest).

#### Windows

Download and install via setup wizard (`markdownpp-setup.exe`) or portable version (`markdownpp-portable.exe`) from the [release page](https://github.com/nlstone/Markdown-PlusPlus/releases/latest).

#### Linux

Download the AppImage, deb, or rpm package from the [release page](https://github.com/nlstone/Markdown-PlusPlus/releases/latest).
