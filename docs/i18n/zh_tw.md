### 功能

- 即時預覽（WYSIWYG）和簡單明瞭的界面，使您獲得無干擾的寫作體驗。
- 支援 [CommonMark Spec](https://spec.commonmark.org/0.29/) 和 [GitHub Flavored Markdown Spec](https://github.github.com/gfm/).
- Markdown擴充功能，例如數學表達式（KaTeX）、front matter 和 emoji。
- 支援段落以及行内樣式的快捷方式，提高您的寫作效率。
- 可以輸出 **HTML** 和 **PDF** 文件。
- 各種主題：**Cadmium Light**、**Material Dark** 等等。
- 各種編輯模式：**原始碼模式**、**打字機模式**、**專注模式**。
- 直接從剪貼簿中貼上圖片。

## 為什麼要另外寫一個編輯器？

1. 我熱愛寫作。我曾經用過很多 Markdown 編輯器，但沒有一個編輯器可以滿足我的需求。我不喜歡在我寫作的時候被莫名其妙的 bug 所干擾。
**MarkDown++** 使用 virtual DOM 來呈現畫面，擁有高效能和開源的附加優勢。任何喜歡 Markdown 和寫作的人都可以使用 MarkDown++。
2. 承上所述，**MarkDown++** 是完全免費和開源的，且將永遠開源。我們希望所有 Markdown 愛好者都可以協助開發並貢獻自己的程式，讓 **MarkDown++** 成為流行的 Markdown 編輯器。
3. Markdown 編輯器很多，各有不同的優點與功能。我們很難滿足所有 Markdown 使用者的需求，但是我們希望 **MarkDown++** 盡可能滿足每一個 Markdown 使用者的需求。儘管最新的 **MarkDown++** 仍不完美，但我們會盡力使它更加完善。

#### macOS

您可以從 [release page](https://github.com/markdownpp/markdownpp/releases/latest) 下載最新的 `markdownpp-%version%.dmg` 或是使用 [**homebrew cask**](https://github.com/caskroom/homebrew-cask) 安裝 MarkDown++。如果使用 Homebrew-Cask，您只需要安裝 [Homebrew](https://brew.sh/)。

```bash
brew install --cask mark-text
```

#### Windows

只需要透過安裝精靈 (`markdownpp-setup-％version％.exe`）下載並安裝 MarkDown++，然後選擇為單一使用者或是所有使用者安裝。

或者可以使用 Chocolatey 或 Winget 等套件管理器來安裝 MarkDown++。

- 使用 Chocolatey 前需要安裝好 [Chocolatey](https://chocolatey.org/install)。

```bash
choco install markdownpp
```

- 使用 Winget 前需要安裝好 [Winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/#install-winget)。

```bash
winget install markdownpp
```

#### Linux

請依照 [Linux installation instructions](../../docs/LINUX.md) 的說明進行安裝。

#### 其它

您可以從以下網址下載 Linux、macOS 和 Windows 的安裝檔：[release page](https://github.com/markdownpp/markdownpp/releases/latest)。如果該版本在您的系統中無法正常使用，請開一個 [issue](https://github.com/markdownpp/markdownpp/issues)。
